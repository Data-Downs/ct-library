# My Library

A personal book library app to catalog, rate, and manage your collection.

## Usage

Open `index.html` in any web browser. No server or build tools needed.

## Features

- Browse books in a responsive card grid
- Add, edit, and delete books
- Track reading status (Unread / Reading / Read)
- Rate books 1-5 stars
- Add personal notes
- Search by title or author
- Filter by status and genre
- Sort by title, author, rating, date, or status
- Export/import your library as JSON
- All data saved locally in your browser (localStorage)
